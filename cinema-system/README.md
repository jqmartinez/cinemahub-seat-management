# 🎬 CinemaHub: Cinema Seat Management System

## 📌 Description
CinemaHub is a seat booking system that allows users to view available seats, select seats, and book tickets. It prevents double booking and manages movies, showtimes, and seat availability.

- Booking system
- Real-time seat availability
- Booking reports

## 🔄 System Flow (Overall)

1. Admin adds movie and showtime  
2. System generates seats  
3. User selects movie and showtime  
4. User selects available seats  
5. System locks seats (no double booking)  
6. User confirms booking  
7. System saves booking and updates seat status
